# Changelog

All notable changes to DevDeck will be documented in this file.

## [0.1.0] - 2026-03-02

### Added
- Initial DevDeck MVP extension scaffold with VS Code sidebar webview.
- Large generated command catalog (2100+ commands across 21 tools).
- Fuzzy search, project-aware suggestions, favorites, and history.
- Parameterized command cards with copy/run actions and keyboard navigation.
- Contributor recognition automation and CI workflow.
